sakila 프로젝트

  • Model 2

Model 2 기법으로 만든 비디오 대여점 관리 프로젝트로 Mysql 샘플 SAKILA 데이터베이스를 분석 수업 후 일주일간 진행한 미니프로젝트입니다. Model 2 기법으로 만들면서 확실히 담당하는 카테고리별로 Controller, Model, View가 나뉘면서 중복되는 코드가 많이 사라지고, 유지보수가 많이 용이해지는 것을 체감했습니다. 또한 sakila DB가 MySQL 샘플 DB이면서도 함수, 프로시저, 트리거, 뷰와 같은 여러 기능을 담고 있어서 DB의 기능들에 대해 자세히 알아보는 계기가 되었습니다.
